# Strategic Copilot Frontend

Frontend of a strategic decision support system with AI Copilot.

## Features
- Decision-first UX (Continue / Adjust / Stop)
- AI Copilot panel with explanations
- Decision timeline & replay
- What-if simulation
- Contextual panels (projects, users, tasks)
- Premium micro-interactions

## Tech Stack
- React / TypeScript
- Vite
- Tailwind

## Goal
Transform complex multi-agent analysis into clear, actionable decisions.
